О сайте

Этот сайт создан с целью тренировки навыков разработки веб-приложений на django. Здесь представлена реализация сайта для изучения испанского языка.

В шапке сайта вы можете найти ссылку на страницу "Словарь", где содержится список слов для заучивания. Для добавления новых слов в словарь нажмите "Добавить слово". Информация об уроках представлена на странице "Уроки", там же там можно найти рекомендации для самостоятельной работы дома. Онлайн консультации с преподавателем будут проводиться раз в две недели, как правило по субботам в 16:00, информация будет размещена на странице "Новости".
